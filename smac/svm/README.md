# Dataset download
Download [CIFAR-10](https://www.cs.toronto.edu/~kriz/cifar-10-python.tar.gz)
and place it in `./data` directory. The end result should look like this
```
data
├── batches.meta
├── data_batch_1
├── data_batch_2
├── data_batch_3
├── data_batch_4
├── data_batch_5
├── readme.html
└── test_batch

1 directory, 8 files
```
